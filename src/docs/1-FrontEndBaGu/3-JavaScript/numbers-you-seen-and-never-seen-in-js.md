# JS 中见过的没见过的数字

<article-info/>

::: code-group

```js
/**
 * 11        11
 * .11       0.11
 * 11.       11
 * 011       9            0开头，按照8进制处理
 * 080       80           0开头，因为有超过8进制的数字，按照10进制处理
 * 0o11      9            0o开头，一定按照8进制处理
 * 0o80      报错          0o开头，一定按照8进制处理，因为有超过8进制的数字，报错
 * 0b11      3            0b开头，一定按照2进制处理
 * 0x11      17           0x开头，一定按照16进制处理
 * 11e2      11*10^2      科学计数法
 * 11.toString()     报错     JS认为.后的应该是数字，但出现了字符，报错
 * 11 .toString()     '11'     JS认为是将十字转换为字符串
 */
```

:::
