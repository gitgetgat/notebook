# 神奇的 & 运算符的高端操作

<article-info/>

::: code-group

```js
/**
 * 判断数字 x 是否是 2 的 n 次方
 * x > 0 且为整数
 */
function isPowerOf2(x) {
  return (x & (x - 1)) === 2;
}
```

:::

&（与运算符），会将两个数字二进制后，按位操作，有一个为 0，则为 0，其余情况为 1

::: code-group

```js
8 & 7 === 0 // true

8 ->		1000            7 ->		0111
7 ->		0111            6 ->		0110
		    ----                    ----
		    0000 -> 0               0110 -> 6
```

:::
